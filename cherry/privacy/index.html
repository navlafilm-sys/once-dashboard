CHERRY CLUB(이하 "운영자")은 인스타그램 계정 @cherry.clubs2 에서 댓글·메시지 자동 응답 서비스를 운영합니다. 본 방침은 해당 서비스가 이용자의 정보를 어떻게 처리하는지 설명합니다.

1. 수집하는 정보

이용자가 먼저 댓글을 달거나 메시지를 보낸 경우에 한해, Meta(Instagram)가 제공하는 공식 API를 통해 아래 정보를 전달받습니다.

항목내용
Instagram 사용자 식별자Instagram이 발급하는 익명 ID(IGSID) 및 공개 사용자명
댓글 내용자동 응답 키워드 판별을 위한 댓글 텍스트 및 댓글 ID
메시지 내용이용자가 보낸 DM 텍스트 (수신거부 요청 확인 목적)
이름, 연락처, 이메일, 주소, 결제정보 등 별도의 개인정보는 일절 수집하지 않습니다. 광고 목적의 프로파일링이나 제3자 판매도 하지 않습니다.

2. 이용 목적

  • 이용자가 요청한 상품 정보·링크를 DM으로 안내
  • 동일 이용자에게 중복 발송을 막기 위한 발송 이력 확인
  • 수신거부(옵트아웃) 요청 처리

3. 보관 및 파기

데이터보관 기간

댓글·메시지 본문은 저장하지 않고, 키워드 판별 직후 폐기합니다. 이용자가 인스타그램에서 메시지를 삭제하면 운영자가 보관한 사본도 삭제됩니다.

4. 제3자 제공 및 처리 위탁

수집한 정보를 제3자에게 판매·제공하지 않습니다. 다만 서비스 운영을 위해 아래 인프라를 이용합니다.

  • Meta Platforms (Instagram) — 댓글·메시지 송수신 API
  • Cloudflare — 자동 응답 서버 및 중복 방지 데이터 저장

5. 수신거부 (Opt-out)

자동 메시지 대화창에서 "그만" 이라고 보내주시면 즉시 발송이 중단되며, 이후 자동 메시지를 보내지 않습니다. 모든 자동 메시지 하단에도 이 안내가 포함됩니다.
